How to Install

Takes about 60 seconds. No account or sign-up required.

1
Download the ZIP from the GitHub repo and unzip it to a folder on your computer.
2
Open Chrome and go to chrome://extensions. Enable Developer Mode in the top right.
3
Click Load unpacked and select the unzipped folder. The extension will appear in your toolbar immediately.
4
Go to claude.ai and start a conversation. The usage meter will appear automatically.
